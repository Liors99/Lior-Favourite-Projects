Bomberman

About the project

  • Was my final project for CPSC 233. I was in a group of 4 people.
  • Fully functional revision of the Bomberman game, which allows up to 2 players to play (on the same computer) or against an AI.
  • Includes full animations for bomb explosion and player movement.
  • Includes powerups the player can pick up to give himself and advantage.
  • Implemented in Java and uses the Swing Java library.
  • Features an AI that uses BFS to track down the player.
