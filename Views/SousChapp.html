Sous-Chapp

About the project

  • Was my final project for CPSC 481. I was in a group of 3 people.
  • A GUI based application designed to provide home cooks with step-by-step instructions on how to cook various recipes using simple and intuitive interface design.
  • Used Visual Studio, and it was Implemented using C# and WPF.
  • Included the entire process of interaction design, from early low fidelity prototypes, which included sketches, to horizontal prototypes and high fidelity prototypes.
