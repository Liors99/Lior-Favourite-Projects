Battleship Game

About the project

  • A console-based application about the popular battleship game.
  • Uses network sockets to communicate with other clients on the same network, allowing them to play each other using TCP connection, developed in Java.
  • Features a client, a server, and an observer.
  • It also includes registration and login with the server.
